    "Description 02:Installing VS Code Extension"::

	A VS Code extension is available for B4P in form of an <b>.vsix</b> file.  The extension supports the following:

	* Color visualization of B4P code in the editor to improve ergonomics
	* Starts B4P by pushing the start button (code will be saved and B4P starts immediaely).

	The VS Code Extension is available on github: <b><a target="_blank" href="https://github.com/georgzurbonsen/B4P_VSCode/">github.com/georgzurbonsen/B4P_VSCode/</a></b>.
	Downlaod the extension file and install it as instructed when opening this web page.
